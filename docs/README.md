# Agentic AI Career Counseling Companion

## Overview

The **Agentic AI Career Counseling Companion** is an AI-powered
application built using **IBM watsonx Orchestrate**, **IBM Granite**,
**Spring Boot**, and **React.js**. It provides personalized career
guidance, resume analysis, skill gap identification, certification
recommendations, and customized learning roadmaps.

## Problem Statement

Students often struggle to choose the right career because existing
career guidance platforms provide generic recommendations. This project
leverages Agentic AI to analyze a student's education, skills,
interests, and resume to deliver personalized career recommendations.

## Features

-   AI-Powered Career Counseling
-   Resume Analysis
-   Skill Gap Identification
-   Career Recommendations
-   Learning Roadmap Generation
-   Certification Suggestions
-   Career Knowledge Base
